## Getting Started

### Installation

```bash
npm install nebula-ui
```

### Basic Usage

```javascript
import { Button, Card } from 'nebula-ui';

function App() {
  return (
    <div>
      <Button variant="primary">Click Me</Button>
      <Card>
        <h2>Welcome to Nebula UI</h2>
        <p>Start building beautiful interfaces today!</p>
      </Card>
    </div>
  );
}
```
